- ポケモン: メガマフォクシー
- 性格: おくびょう (素早さ上昇・攻撃下降)
- 特性: ふゆう
- レベル: 50
- 持ち物: なし
- 天候: なし
- フィールド: なし
- テラスタル: なし
- 急所: なし

## 2. 比較する3つの努力値配分案

いずれも素早さ努力値29 (素早さ201、メガゲンガーの素早さ200を抜く設計) とHP努力値3 (HP153、みがわり4回分確保) は共通。残り努力値を防御と特攻にどう分配するかが違う。

## 3. メガマフォクシーが使う3つの技

- **サイコショック**: エスパータイプ、特殊技、威力80。ただしダメージ計算は相手の防御を参照する (特殊技だが防御で判定)。
- **マジカルシャイン**: フェアリータイプ、特殊技、威力80。ダメージ計算は相手の特防を参照する。
- **かえんほうしゃ**: ほのおタイプ、特殊技、威力90。タイプ一致ボーナス1.5倍 (メガマフォクシーはほのおタイプ)。ダメージ計算は相手の特防を参照する。

## 4. 仮想敵の耐久振り方3パターン

各対象ポケモンについて、よく使われる3種類の耐久配分で計算する。

- **無振り**: 努力値すべて0、性格補正なし (メガ進化して直後そのまま)
- **HPのみ振り**: HP努力値を32まで、防御(または特防)努力値は0、性格補正なし
- **耐久特化**: HP努力値32 + 防御(または特防)努力値32 + 性格補正あり
  - 物理耐久特化の性格はずぶとい (防御1.1倍、攻撃0.9倍)
  - 特殊耐久特化の性格はおだやか (特防1.1倍、攻撃0.9倍)

対象ポケモン: メガゲンガー / メガキラフロル / メガフシギバナ / メガハッサム / マスカーニャ / ミミッキュ / サザンドラ / バンギラス / メガガルーラ / イダイトウ / ドドゲザン / メガスターミー / アーマーガア / メガルカリオ / ガブリアス / メガマフォクシー / アシレーヌ (合計17匹)

## 5. タイプ相性と特性による注意事項

- あくタイプを持つマスカーニャ・サザンドラ・バンギラス・ドドゲザンはエスパー技が無効 (0倍)、サイコショックは当たってもダメージゼロ
- メガフシギバナは特性「あついしぼう」でほのお技のダメージを0.5倍に軽減 (かえんほうしゃの最終倍率計算に反映済み)
- 乱数1発とは、16段階の乱数表の中で相手のHPを上回る段階が何段階あるかで判定 (16段階中6段階なら確率37.5パーセント)
