## 2. 比較する3つの努力値配分案

いずれも素早さ努力値29 (素早さ実数値201、メガゲンガーの素早さ実数値200を抜く設計) と体力努力値3 (体力実数値153、みがわり4回分確保) は共通。残り努力値を物理防御と特攻にどう分配するかが違う。

## 3. メガマフォクシーが使う3つの技

- **サイコショック**: エスパータイプ、特殊技、威力80。ただしダメージ計算は相手の物理防御実数値を参照する (特殊技だが物理防御で判定)。
- **マジカルシャイン**: フェアリータイプ、特殊技、威力80。ダメージ計算は相手の特防実数値を参照する。
- **かえんほうしゃ**: ほのおタイプ、特殊技、威力90。タイプ一致ボーナス1.5倍 (メガマフォクシーはほのおタイプ)。ダメージ計算は相手の特防実数値を参照する。

## 4. 仮想敵の耐久振り方3パターン

各対象ポケモンについて、よく使われる3種類の耐久配分で計算する。

- **無振り**: 努力値すべて0、性格補正なし (メガ進化して直後そのまま)
- **体力のみ振り**: 体力努力値を32まで、物理防御(または特防)努力値は0、性格補正なし
- **耐久特化**: 体力努力値32 + 物理防御(または特防)努力値32 + 性格補正あり
  - 物理耐久特化の性格はずぶとい (物理防御1.1倍、攻撃0.9倍)
  - 特殊耐久特化の性格はおだやか (特防1.1倍、攻撃0.9倍)

対象ポケモン: メガゲンガー / メガキラフロル / メガフシギバナ / メガハッサム / マスカーニャ / ミミッキュ / サザンドラ / バンギラス / メガガルーラ / イダイトウ / ドドゲザン / メガスターミー / アーマーガア / メガルカリオ / ガブリアス / メガマフォクシー / アシレーヌ (合計17匹)

## 5. タイプ相性と特性による注意事項

- あくタイプを持つマスカーニャ・サザンドラ・バンギラス・ドドゲザンはエスパー技が無効 (0倍)、サイコショックは当たってもダメージゼロ
- メガフシギバナは特性「あついしぼう」でほのお技のダメージを0.5倍に軽減 (かえんほうしゃの最終倍率計算に反映済み)
- 乱数1発とは、16段階の乱数表の中で相手の体力を上回る段階が何段階あるかで判定 (16段階中6段階なら確率37.5パーセント)
