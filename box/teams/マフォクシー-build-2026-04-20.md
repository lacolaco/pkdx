## 選出パターン

### vs Pattern A 環境標準 (リザY/アーマー/ゲン/カバ/ドド/ギル)

* 選出: ブラッキー + イダイトウ♂ + マフォクシー
* 理由: Nash value -0.325 (v3 -0.005より大劣化)。ブリジュラス消失でアーマー/ゲン対特殊打点不足。ブラッキーあくび回し+Mマフォかえんほうしゃで凌ぐ形。

### vs Pattern B 高速アタッカー (リザY/ゲン/マスカ/ミミロ/サザン/ロトム)

* 選出: ブラッキー + イダイトウ♂ + マフォクシー
* 理由: Nash value -0.410 (v3 +1.0からの大崩壊)。ブリジュラス10まん4×(リザY)/はどうだん4×(サザン)消失、ブラッキーでは特殊高火力に耐久勝ちできない。このパターンは実戦発生しにくいが注意。

### vs Pattern C 受け回し (アーマー/カバ/キラフロ/ロトム/ブラッキー/ギル)

* 選出: ガブリアス + アシレーヌ + マフォクシー
* 理由: Nash value +0.082 (v3 +0.080とほぼ同等)。76%本選出。Mマフォかえんほうしゃ+サイコショックで受け崩し。ブラミラーでは耐久差で差別化できず0%選出。

### vs Pattern D 対マフォクシー特化 (ドド/ギル/ミミ/サザン/ブラッキー/キラフロ)

* 選出: ガブリアス + イダイトウ♂ + ルカリオ
* 理由: Nash value +0.344 (v3 +0.223より+0.121改善)。Mルカリオインファイト+かみくだく+イダイトウおはかまいりでドドゲ/ミミ/ギル崩し、Mマフォ外しでパーティ機能。

## 構築の強み

* 実相手4パターン (E/F/G/H) 平均 +0.095 で v3 (-0.008) から +0.10 改善
* メガ2枠 (Mマフォクシー + Mルカリオ) で全8パターン中 7/8 メガ選出維持
* ブラッキー (ずぶとい H32B32D2) 特殊受け+あくび回し、Mゲンガーシャドボ/マスカはたき等を吸収
* イダイトウ♂ スカーフ AS (てきおうりょく STAB×2) で高速物理打点、対Mルカ/スカガブの上から殴る
* Mマフォクシー わるだくみ+みがわり型で積み展開、みがわり盾で先制技受け流し
* Mルカリオ かみくだく採用でMゲンガー/ミミッキュ/ギルガルド (ゴースト系) に打点
* アシレーヌ (ずぶとい B特化) で物理受け兼 めいそう積み
* ガブリアス (ようき タスキ) ステロ展開役は継続

## 構築の弱み・注意点

* Pattern B (高速アタッカー架空環境) で value -0.41 の大崩壊 - ブリジュラス削除で特殊高火力対策消失
* Pattern A (環境TOP10) -0.325 で大劣化 - アーマーガア/Mゲンガー対特殊打点不足
* ブリジュラス削除により 10まん4×/はどうだん4× の抜群技が消える - 電気/鋼耐性消失
* ブラッキー イカサマ 以外物理攻撃技なし、火力面は期待できない
* イダイトウ♂ スカーフ固定で技ロック、選出前の型バレで不利化
* Mマフォクシーのめいそう→わるだくみ変更でD積みがなくなり、特殊耐久が落ちる
* Mルカリオ Spe 180 だがスカーフガブ (253) には抜かれる
* 平均 Nash value +0.009 は v3 (+0.158) より -0.15 劣化 - メタ想定次第では大幅劣化リスク
