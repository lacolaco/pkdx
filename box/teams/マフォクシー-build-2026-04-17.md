## 選出パターン

### vs Pattern A 環境標準 (リザY/アーマー/ゲン/カバ/ドド/ギル)

* 選出: ガブリアス + マフォクシー + ブリガロン
* 理由: Nash value +0.028 (旧+0.106)。74%本選出。スカガブステロ/げきりん+Mマフォくさむすび+Mブリガロンやどりぎで削り。

### vs Pattern B 高速アタッカー (リザY/ゲン/マスカ/ミミロ/サザン/ロトム)

* 選出: ガブリアス + マフォクシー + ブリガロン
* 理由: Nash value +1.0 完全勝利。メガ2枚フル選出でも完封 (旧はメガ腐りだったがこちらは両メガ選出)。

### vs Pattern C 受け回し (アーマー/カバ/キラフロ/ロトム/ブラッキー/ギル)

* 選出: ブリジュラス + アシレーヌ + ガブリアス
* 理由: Nash value +0.059。メガ2枚とも腐るPattern。非メガ3体でじわ削り。

### vs Pattern D 対マフォクシー特化 (ドド/ギル/ミミ/サザン/ブラッキー/キラフロ)

* 選出: ガブリアス + マフォクシー + ブリガロン
* 理由: Nash value +0.184 (旧と同等)。対マフォ環境でもMブリガロンのボディプレスでドドゲ/ブラ対策+Mマフォでゴースト崩し。メガ2枚選出達成。

### vs Pattern E 実相手1 (MリザX/ブリ/スカガブ/マスカ/アシ/Mハッサム)

* 選出: ガブリアス + マフォクシー + ブリガロン
* 理由: Nash value -0.039 (旧+0.001)。スカーフガブ先制でアドバンテージ。Mブリガロンでマスカーニャのタネばくだん等を無効化 (ぼうだん)。

### vs Pattern F 実相手2 (ガルーラ軸) (Mガルーラ/アーマー/ロトムW/タスキガブ/ミミ/黒眼鏡ドド)

* 選出: ブリジュラス + アシレーヌ + ブリガロン
* 理由: Nash value -0.081 (旧-0.050)。Mブリガロンで物理受け+やどりぎ削り、対Mガルーラおやこあいの耐久勝負。

### vs Pattern G 実相手3 (アーマー/キラフロル/スカガブ/ミミ/Mゲン/MリザY)

* 選出: ガブリアス + マフォクシー + ブリガロン
* 理由: Nash value +0.100 (旧+0.117)。スカーフガブでスカガブ抜き合戦 + Mマフォサイコキネシスでメガゲン、Mブリガロン(ぼうだん)でシャドボ無効。

### vs Pattern H 実相手4 (MリザY/Mルカリオ/カバ/ロトムW/スカガブ/ミミ)

* 選出: アシレーヌ + ガブリアス + マフォクシー
* 理由: Nash value +0.089 (旧+0.171)。Mルカリオ先制手段弱体化、Mマフォかえんほうしゃでリザ/ルカ対応、スカガブでスピ制圧。

## 構築の強み

* メガ2枠 (Mマフォクシー + Mブリガロン) で Pattern C 以外 7/8 Pattern で100%メガ選出達成
* メガ腐り問題ほぼ解消 (旧チームでは Pattern B/D で 0%腐り、改訂版で100%選出)
* Mブリガロン ぼうだん特性 でボール/爆弾系技 (シャドーボール/きあいだま/タネばくだん 等) 完全無効
* Mブリガロン 物理受け+ボディプレス でドドゲザン/ガブ/Mガルーラに耐久勝ち
* スカーフガブリアス (S=253相当) で環境の高速帯を制圧 (Mルカリオ/Mゲンガー以外抜ける)
* マフォクシー めいそう型採用で対耐久受け (アシ/ロトム/ブラッキー) に積み崩し
* マスカーニャ へんげんじざい (トリックフラワー必中急所 + ちょうはつ) で積み/受け崩し
* 平均Nash value +0.167 (旧+0.202比 -0.035のみ劣化) でバランス維持

## 構築の弱み・注意点

* Pattern C (受け回し) で両メガ選出率 0% - 非メガ 3 体で削る持久戦、構造的弱点
* Mハッサム バレットパンチ先制技消失 - Pattern H のMルカリオ急襲に弱い (+0.089 で旧より-0.082)
* カイリュー しんそく先制技消失 - 対マスカ/ミミ の先手処理手段が減少
* ドドゲザン そうだいしょう発動下で Mマフォ 即死 (ふいうち 確1) は旧と同じ
* スカーフガブ 技固定リスク - げきりんロック時に一貫された相手 (フェアリー) にスイッチ誘発
* Mブリガロン Spe 44 極低 - 先手取れない相手多数、先制技 (ふいうち/しんそく) で詰む
* マスカーニャ タスキ前提 - ステロ展開済みだと 1 回耐え保証が崩れる
* アシレーヌ B特化調整で特殊耐久低下 - 強特殊 (Mゲン シャドボ 等) で持ちが悪い
