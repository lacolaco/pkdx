## 選出パターン

### vs Pattern A 環境標準 (リザY/アーマー/ゲン/カバ/ドド/ギル)

* 選出: ガブリアス + マフォクシー + ルカリオ
* 理由: Nash value -0.005 (旧+0.106)。68%本選出。ガブ先発でステロ展開+Mマフォかえんほうしゃ+Mルカリオインファイトでドドゲ対策。

### vs Pattern B 高速アタッカー (リザY/ゲン/マスカ/ミミロ/サザン/ロトム)

* 選出: ガブリアス + ブリジュラス + アシレーヌ
* 理由: Nash value +1.0 完全勝利維持。非メガ3体で完封、メガは温存。

### vs Pattern C 受け回し (アーマー/カバ/キラフロ/ロトム/ブラッキー/ギル)

* 選出: ブリジュラス + アシレーヌ + マフォクシー
* 理由: Nash value +0.080 (v1 +0.059 より改善)。63%本選出。Mマフォねっさのだいちでキラフロル確1+アシめいそう積み+ブリ10万でアーマー粉砕。

### vs Pattern D 対マフォクシー特化 (ドド/ギル/ミミ/サザン/ブラッキー/キラフロ)

* 選出: ガブリアス + アシレーヌ + ルカリオ
* 理由: Nash value +0.223 (旧+0.184より大幅改善)。96%本選出。Mルカリオインファイトでドドゲ確1、ブラ2×、アシムンフォでサザン4×、Mマフォは選出外。

### vs Pattern E 実相手1 (Mリザ+ブリ+スカガブ) (MリザX/ブリ/スカガブ/マスカ/アシ/Mハッサム)

* 理由: Nash value -0.111 (旧+0.001から大きく悪化)。相手Mハッサムミラー勝ち筋消失。ロトムスカーフトリックでスカガブ無力化+Mルカリオで削り。

### vs Pattern F 実相手2 (ガルーラ軸) (Mガルーラ/アーマー/ロトムW/タスキガブ/ミミ/黒眼鏡ドド)

### vs Pattern G 実相手3 (Mリザ+Mゲンガー) (アーマー/キラフロル/スカガブ/ミミ/Mゲン/MリザY)

## 構築の強み

* メガ2枠 (Mマフォクシー + Mルカリオ) で Pattern B 以外 7/8 で 100% メガ選出
* Pattern D (対マフォクシー特化) が +0.223 で改訂前+0.039改善 - Mルカリオインファイトがドドゲ/ブラ抜群
* ロトム (ウォッシュ) スカーフトリックでスカガブ/Mガルーラの型封じ (Pattern E/F/G で100%選出)
* マフォクシー技更新 (サイコショック/ねっさのだいち) で対特殊受け (ブラッキー等) + 対毒/鋼 (キラフロル/ドドゲ) 火力追加
* ガブリアス タスキ+ステロで展開役復活、ブリジュラスもステロ持ちで二重展開可能
* Mルカリオ てきおうりょく STAB 2倍でインファイト/バレットパンチ実効威力UP
* アシレーヌ B特化で物理受け+めいそう積み堅持

## 構築の弱み・注意点

* 【総合】Nash 平均 +0.158 (旧Mハッサム軸+0.202より -0.044 劣化)
* 【Pattern E】相手Mハッサム対策消失で -0.112 大劣化 - バレパン先制による2×打点失う
* 【Pattern G】Mハッサム対MリザY(4×)/Mゲンガー(2×)消失で -0.095 劣化
* 【Pattern A】-0.111 - Mハッサム先制バレパンの安定性喪失
* スカガブ対応は改善（ロトムトリック）だが、スカガブ自身の抜きは通される (ロトムS<253)
* ドドゲザンそうだい発動下でMマフォ即死は v1 と同じ - 先処理必須
* Mルカリオ こおり 4倍弱点 (アローラキュウコン ふぶき確1級)
* Pattern B ではメガなし選出 - 完封勝ちだがメガ腐り
