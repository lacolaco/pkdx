## 選出パターン

### vs Pattern A 環境標準 (リザードンY/アーマーガア/ゲンガー/カバルドン/ドドゲザン/ギルガルド)

* 選出: マフォクシー + カイリュー + ハッサム
* 理由: Nash value +0.106。54.5%でこの選出+45.5%でガブ/ブリ/マフォ。ドドゲザンそうだいしょう発動前に他メンバーで処理、MマフォクシーはリザY/カバルドンへの崩し担当。

### vs Pattern B 高速アタッカー (リザードンY/ゲンガー/マスカーニャ/ミミロップ/サザンドラ/ロトム)

* 選出: ガブリアス + ブリジュラス + アシレーヌ
* 理由: Nash value +1.0(完全勝利)。高速相手に対しブリ耐久+4倍打点、アシめいそう積み、ガブ先制スカーフ抜きで完封。

### vs Pattern C 受け回し (アーマーガア/カバルドン/キラフロル/ロトム/ブラッキー/ギルガルド)

* 選出: ブリジュラス + アシレーヌ + カイリュー
* 理由: Nash value +0.090。71.2%で選出。カイリューりゅうまい+しんそくで全抜き、ブリ10万でアーマーガア、アシめいそうで受け崩し。

### vs Pattern D 対マフォクシー特化 (ドドゲザン/ギルガルド/ミミッキュ/サザンドラ/ブラッキー/キラフロル)

* 選出: ガブリアス + アシレーヌ + カイリュー
* 理由: Nash value +0.184。マフォクシーは腐るので外す。アシムンフォがあく4体に抜群、ガブじしん、カイリュー先制で制圧。

### vs Pattern E 実対戦相手(スクショ取り込み) (メガリザX/ブリ/スカーフガブ/マスカーニャ/アシ/メガハッサム)

* 選出: ガブリアス + ブリジュラス + ハッサム
* 理由: Nash value +0.0006(完全五分)。スカーフガブにカイリュー抜かれ、Mマフォクシーはアシ/スカーフガブに不利。ブリ10万ボルトでMハッサム抜群、ハッサムでマスカーニャ牽制。

## 構築の強み

* Nash 5パターン全てで value > 0。現環境全域で戦える構築
* アシレーヌ (出番2.71) がメタ全体に刺さる主軸 - ムンフォ4×(あく/ドラゴン) + めいそう積み
* カイリュー (2.54) のしんそく先制が高速アタッカー (Pattern B/D) を封殺
* ブリジュラス はどうだん 4× (あく/はがね) でドドゲザンを高確率処理、10まん4×でアーマーガア粉砕
* Mマフォクシー くさむすび + ガブステロで物理受け3体 (カバルドン/キョジオーン/マンムー) を確1圏
* Mハッサム バレットパンチがメガゲンガー (ゴースト/どく) に抜群で即死級

## 構築の弱み・注意点

* ドドゲザン そうだいしょう発動下で Mマフォクシー 即死 (ふいうち 確1) - 先処理必須
* Pattern D (対マフォクシー特化) では Mマフォクシー 選出不能 - メガ枠腐るリスク
* スカーフガブリアス (S=253) に抜かれる - ガブ/ブリ/カイ の上から げきりん 4× 級被弾
* マスカーニャへんげんじざい の先制 ふいうち + タスキ で エース崩し
* サイコキネシス が相手あくタイプ (ドドゲ/ブラッキー/ゲッコウガ) に無効 - かえんほうしゃ依存
* メガマフォクシーのアンコール は Pattern A の積み技持ち (リザY剣舞/ギル剣舞等) には機能、Doduge には無力
