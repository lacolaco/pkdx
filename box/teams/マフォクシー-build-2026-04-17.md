## 選出パターン

### vs Pattern A 環境標準 (リザY/アーマー/ゲンガー/カバ/ドド/ギル)

* 選出: マフォクシー + カイリュー + ハッサム
* 理由: Nash value +0.106。54.5%でこの選出+45.5%でガブ/ブリ/マフォ。ドドゲザンそうだいしょう発動前に他メンバーで処理、MマフォクシーはリザY/カバルドンへの崩し担当。

### vs Pattern B 高速アタッカー (リザY/ゲンガー/マスカ/ミミロ/サザン/ロトム)

* 選出: ガブリアス + ブリジュラス + アシレーヌ
* 理由: Nash value +1.0(完全勝利)。高速相手に対しブリ耐久+4倍打点、アシめいそう積み、ガブ先制スカーフ抜きで完封。

### vs Pattern C 受け回し (アーマー/カバ/キラフロ/ロトム/ブラッキー/ギル)

* 選出: ブリジュラス + アシレーヌ + カイリュー
* 理由: Nash value +0.090。71.2%で選出。カイリューりゅうまい+しんそくで全抜き、ブリ10万でアーマーガア、アシめいそうで受け崩し。

### vs Pattern D 対マフォクシー特化 (ドド/ギル/ミミ/サザン/ブラッキー/キラフロ)

* 選出: ガブリアス + アシレーヌ + カイリュー
* 理由: Nash value +0.184。マフォクシーは腐るので外す。アシムンフォがあく4体に抜群、ガブじしん、カイリュー先制で制圧。

### vs Pattern E 実相手1(Mリザ+ブリ+スカーフガブ軸) (メガリザX/ブリ/スカガブ/マスカ/アシ/メガハッサム)

* 選出: ガブリアス + ブリジュラス + ハッサム
* 理由: Nash value +0.0006(完全五分)。スカーフガブにカイリュー抜かれ、Mマフォクシーはアシ/スカーフガブに不利。ブリ10万ボルトでMハッサム抜群、ハッサムでマスカーニャ牽制。

### vs Pattern F 実相手2(ガルーラ軸) (メガガルーラ/アーマー/ロトムW/タスキガブ/ミミ/黒眼鏡ドド)

* 選出: アシレーヌ + カイリュー + ハッサム
* 理由: Nash value -0.050(不利)。メガガルーラおやこあいじしん+ふいうちで自軍エースが落ちやすい。アシめいそうで耐久勝負、カイリューしんそくでミミッキュ/ガルーラ仕留め、ハッサムバレパンで詰ませる。マフォクシー/ブリは選出圏外。

### vs Pattern G 実相手3(Mリザ+ゲンガー軸) (アーマー/キラフロル/スカガブ/ミミ/メガゲンガー/MリザY)

* 選出: ガブリアス + マフォクシー + ハッサム

## 構築の強み

* Nash 7パターン中 6 パターンで value > 0。ほぼ全環境で戦える
* アシレーヌ(出番3.26)が最頻出主軸 - ムンフォ4×(あく/ドラゴン) + めいそう積み
* カイリュー(3.00)のしんそく先制が高速アタッカー(Pattern B/D/F)を封殺
* ブリジュラス はどうだん 4×(あく/はがね)でドドゲザン高確率処理、10万4×でアーマーガア粉砕
* Mマフォクシー くさむすび+ガブステロで物理受け(カバ/キョジオ/マンムー)を確1圏
* Mハッサム バレパンがメガゲンガー(どく)に抜群、メガリザY(むし4× はがね2×)に確1
* 7パターンの平均ゲーム値 +0.22 → 有利

## 構築の弱み・注意点

* Pattern F (メガガルーラ+タスキガブ型)で value -0.050 の唯一の不利環境
* メガガルーラ おやこあい は Mマフォクシー等のエースを2ヒットで即落とす - ブリも含め物理高耐久必要
* ドドゲザン そうだいしょう発動下で Mマフォクシー 即死(ふいうち 確1) - 先処理必須
* スカーフガブリアス(S=253)に抜かれる - ガブ/ブリ/カイの上から げきりん 4× 被弾
* マスカーニャ へんげんじざい の先制 ふいうち + タスキでエース崩し
* サイコキネシスが相手あくタイプ(ドド/ブラ/ゲッコウガ)に無効 - かえんほうしゃ依存
* Pattern D(対マフォクシー特化)では Mマフォクシー 選出不能 - メガ枠腐るリスク
