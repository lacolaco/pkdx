* メガ2枠 (Mマフォクシー + MリザードンX) でメガ腐り率を下げる構造 - 理論上のメリット
* ドドゲザン メタルバースト型でカウンター + 剣舞積み性能
* Mマフォクシー (特殊軸) と MリザードンX (物理軸) の両刀メガでタイプ幅広
* ブリジュラス C振り調整でC 176 → 特殊火力アップ (対アーマー/ロトム 効率改善)
* アシレーヌ B特化 (Def138/A↓) で物理受け性能向上 - スカガブどくづき 0.5× 耐え
* Pattern B 完全勝利 (+1.0)、Pattern D 互角 (+0.183) は Team1 と同等

## 構築の弱み・注意点

* 【総合】Nash 平均勝率 +0.101 (Team1+0.202 より -0.101 劣化) → 全環境で Team1 に劣る
* 【致命】カイリュー 不在でしんそく先制技消失 - マスカーニャ/ミミッキュ/ガルーラ等の高速物理を止められない
* 【致命】Mハッサム 不在でバレパン先制消失 - 対MリザY(4×)/Mゲンガー(2×)の決定打喪失 → Pattern E/G が崩壊
* 【Pattern E 実相手1】value -0.247 で大不利 - 相手Mハッサムミラーの優位も失う
* 【Pattern G 実相手3】value -0.154 で大不利 - 相手Mゲンガー/MリザY処理不能
* 【Pattern H】value +0.035 (Team1 +0.171) - Mルカリオ急襲への先制手段欠如
* 【Mリザ X 腐り】Nash 選出は Pattern A の 15% のみ - スカガブ/Mルカリオに先手取られ起動困難
* 【ドドゲザン依存】Pattern C-G で 100% 選出だがそうだいしょう発動前は火力不足 + 低速 S63で後手
