# Champions M-1 / M-A レギュレーション制約メモ

**目的**: 提案時に使用不可アイテム・未習得技を提示するミスを防ぐ。**提案前に必ず参照すること**。

## 原則

1. **観測済みのみ信用**: 自チーム・相手チーム・参考チーム (yakkun 等) で実際に使われた物のみ「使用可」と判定
2. **過去 SV 等の知識を流用しない**: Champions は独自レギュレーションで、アイテム・技・メガの使用可否が異なる
3. **不明なら DB 照合**: `pkdx moves`, `pkdx query`, `pokedex.db` SQL で個別確認
4. **ルール違反提案は NG**: 「代替案としてハチマキ」のような曖昧提案は避け、M-1 内で完結させる

## 【使用可】観測済みアイテム

### 耐久系
- **たべのこし** (ブリジュラス/ブラッキー/カバルドン/デスカーン等で観測)
- **オボンのみ** (アシレーヌ/デスバーン等)
- **ラムのみ** (カイリュー/ドドゲザン/アシレーヌ等)
- **しろいハーブ** (ブリジュラス等)

### タスキ・攻撃系
- **きあいのタスキ** (ガブリアス/イダイトウ/マスカーニャ/デスカーン等)
- **こだわりスカーフ** (ガブリアス/マスカーニャ/ロトム(W)/イダイトウ等) ← **ハチマキ無しだがスカーフは有り**

### タイプ強化系
- **くろいメガネ** (ドドゲザン等) ← あく技 1.2×
- **ようせいのハネ** (ミミッキュ等) ← フェアリー技強化 (要再確認)
- **のろいのおふだ** (ミミッキュ等) ← ゴースト技強化

### 防御系
- **ゴツゴツメット** (アーマーガア等)
- **ひかりのこな** (ミミッキュ等) ← 回避率↑

### メガ石
- 全 59 種 (Champions DB 登録済み) - 別記

## 【使用不可】確定で M-1 に無いアイテム

ユーザーから明示的に指摘されたもの:

- ✕ **こだわりハチマキ**
- ✕ **こだわりメガネ**
- ✕ **いのちのたま**
- ✕ **とつげきチョッキ**

これらは過去作品 (SV 等) の知識で提案しがちだが **M-1 には存在しない**。絶対に提案に含めない。

## 【習得不可】確認済み技制約

Champions 版では canonical と異なり、一部の技を**習得できないポケモン**あり:

| ポケモン | canonical 習得 | Champions で **不可** |
|---|---|---|
| ドドゲザン | はたきおとす、ばかぢから | ✕ 両方不可 |
| メガハッサム | (canonical と同じ) | (特記なし) |
| マスカーニャ | シザークロス、とびげり、はたきおとす | ✕ 全て不可 |
| ミミロップ | しねんのずつき | ✕ 不可 |
| ロトム | (フォーム特定技は本体に無い) | ✕ ハイドロポンプは洗ロトム限定 |
| アローラキュウコン | - | DB には居るが CLI name-引き不可 (region フィルタ) |

→ **技提案前に `pkdx moves <name> --version champions` で毎回確認** が確実。

## 特殊な仕様メモ

### 特性の Champions 独自
- **メガメガニウム**: ability1 = **メガソーラー** (canonical と異なる)
- **メガミミロップ**: メガランチャー ではなく `じゅうなん` 系?要 DB 確認
- **メガボーマンダ**: DB に存在しない (使用不可)

### Mega 特性変更
- 通常 Lucario は せいしんりょく → Mega で **てきおうりょく** (STAB ×2)
- ギャラドス いかく → Mega ギャラドス **かたやぶり** (Champions 特殊、canonical Mold Breaker 同義)
- リザードンY もうか → **ひでり**

### Champions SP 仕様 (既知)
- SP 最大 32/各 stat、合計 66
- HP = base + SP + 75 (他 = floor((base + SP + 20) × nature_mod))
- EV/IV 廃止、--version scarlet_violet のみで EV 計算

## 提案前チェックリスト

アイテム・技・メガ・特性を提案する時、以下を確認:

1. **アイテム**: このファイルの「観測済み」か「NG 一覧」で判定。不明なら「確認要」と明記して提案しない
2. **技**: `pkdx moves <pokemon> --version champions` で習得確認
3. **メガ**: 上記 59 リストに含まれるか確認 (含まれない = 提案不可)
4. **特性**: `pkdx query <name>` で `ability1/ability2/dream_ability` を確認。メガ後特性は DB 直接 (`local_pokedex_ability WHERE mega_evolution IS NOT NULL`) で確認

**過去知識 (SV / USUM / 剣盾) をそのまま流用しない**。Champions は全く別レギュレーション。
